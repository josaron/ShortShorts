## Replacing with Real Music

For production use, replace these placeholder files with actual royalty-free music from:

- [Pixabay Music](https://pixabay.com/music/) - CC0 licensed
- [Free Music Archive](https://freemusicarchive.org/) - Various CC licenses
- [Bensound](https://www.bensound.com/) - Free with attribution
- [Incompetech](https://incompetech.com/) - CC BY license

### Recommended Specifications

- Format: WAV or MP3
- Bitrate: 192kbps or higher for MP3
- Duration: 90-150 seconds (will loop if needed)
- No vocals (instrumental only)

### License Requirements

Ensure all tracks have a license that permits:
- Commercial use
- Modification
- No attribution required (preferred for simplicity)

## Regenerating Placeholders

To regenerate the placeholder files, run:

```bash
node scripts/generate-music.js
```
